About DesignAssistant

Project Scope

As designers, there is inspiration circling us everywhere. The problem is if we see elements we like or want to use in our own works, there is no place that we can snapshot and upload pictures of what we are inspired by and receive fonts and color codes to create color palettes in real time. This can assist designers everywhere in creating designs utilizing the colors and fonts they drew inspiration from. The possibilities would be endless with an application that provides these services in one compact location.

I believe that a picture uploading based web app that provides color codes and fonts names based on the image will help designers when it comes to creating color palettes and dictating font choices based on outside inspiration.
